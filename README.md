# RFID-pro
### 使用说明

该项目基于VS2017建立，需要在X86下编译运行，需要额外的数据库支持(不提供)。

代码请使用VS2017 15.7.1 及以上版本打开，不考虑向下兼容。

平台工具集为 V141,不保证低于该版本能运行，正在测试intel icc 18的优化方式。
